# 乐谱打点小工具 — 操作手册

本文对应浏览器页面：`http://<服务器>:8000/marker/`（本机示例：`http://127.0.0.1:8000/marker/`）。

**查看已发布曲目（谱面 + 标注 + 试听）**：`http://<服务器>:8000/viewer/`（本机：`http://127.0.0.1:8000/viewer/`），需 App Token，与 App 使用同一套 `/api/scores` 数据。

---

## 一、页面是干什么的

在**谱面静态图**上为每个小节画一个矩形，并把矩形与**音频时间轴**（毫秒）绑定，最后导出 **`data.json`**，供 Android 练习页做高亮与跳转。

- **推荐**：选好谱图与 MP3 后先点 **「上传到服务器」**，由后端暂存并返回 URL，右侧用 HTTP 加载预览（可绕开本机「导入/云占位」等读文件问题）。需 **App Token** 与服务器环境变量 `APP_TOKEN` 一致（默认开发值多为 `dev-app-token-change-me`）。
- **可选**：不点上传时，仍可用本机 **FileReader / blob** 直接预览（大文件或个别环境可能不稳定）。
- 导出的 **`data.json` 里约定文件名**为 `sheet.jpg` 与 `audio.mp3`。通过 **管理页** 上传整套资源时，请与练习包命名一致，或保证目录里 `data.json` 的 `imageFile` / `audioFile` 与实际文件名一致。

---

## 二、推荐准备

| 项目 | 说明 |
|------|------|
| 浏览器 | 推荐 **Chrome** 或 **Edge**（新版）；避免过旧 IE。 |
| 谱面图 | **JPG / PNG** 等常见格式；文件不要过大（建议单张 20MB 内）。 |
| 音频 | **MP3**（与 App 端播放一致）；也可用 MIDI 在浏览器里试听，但导出 JSON 仍按 `audio.mp3` 命名约定，管理端上传 MIDI 时会由服务器转 MP3。 |
| 键盘 | 绑定时间需要按 **空格键**，请保证焦点在页面内（点一下页面空白处再操作）。 |

## 三、分步操作（按顺序做）

### 1. 打开页面

在地址栏输入打点小工具地址，等待页面完整加载。

### 2. 填写 songId 与标题（可选但建议填）

- **songId**：英文/数字/下划线，例如 `santao_che_01`，会写入 JSON，并与服务器上曲谱目录名一致时最省事。
- **标题 title**：显示用，例如「三套车」。

### 3. 选择谱面图片与音频

1. **谱面图片**：点 **「谱面图片」** → 选本地 **JPG / PNG** 等（勿选 **0 字节** 或仅云端的「导入」占位文件）。
2. **音频**：点 **「音频」** → 选与练习一致的 **MP3**（推荐）。

### 4.（推荐）上传到服务器后再打点

1. **App Token**：与服务器 `APP_TOKEN` 一致（开发默认多为 `dev-app-token-change-me`）。
2. **后端根地址**：用 Chrome 直接打开 `http://127.0.0.1:8000/marker/` 时一般可**留空**（与当前页同源）。若用 Cursor 内置预览、端口转发等，请填 **`http://127.0.0.1:8000`**（端口按实际改）。
3. 点击 **「上传到服务器」**，等待状态栏提示成功；右侧应显示谱图，下方播放器可显示时长并可播放。
4. 若异常：先 **Ctrl+F5** 强刷；确认 `openapi.json` 里存在 `/api/marker/upload`；可看折叠 **「调试日志」** 或终端里跑 `server/scripts/playwright_marker_debug.py` 做自动检查。

### 5.（可选）仅用本机预览

不上传时，选好文件后页面会尝试用本机读取显示图与音频；大文件可能走 blob 模式。若右侧仍不显示，请改用上一节 **上传到服务器**。

### 6. 在谱面上画小节框（矩形）

1. 在**右侧谱图**上 **按住鼠标左键不放**，拖动拉出矩形，松开即画好一个「待绑定」的框。  
2. 可连续画多个框；**与音频时间的绑定顺序**是：每次按空格，绑定 **当前最后一个未绑定的框**（先画先绑的逻辑通过「画一个绑一个」最清晰）。  
3. 画错可用 **「撤销上一小节」**：优先撤销已绑定的小节，没有则撤销未绑定的框。

### 7. 用空格把「框」和「时间」绑成一个小节

1. **先播放**音频，在认为**该小节结束边界**的时刻（或该小节最后一个音落点）**暂停**，或保持播放中也可。  
2. **按键盘空格**。  
3. 工具会把 **当前播放时间（毫秒）** 记为这一小节的 **`endTimeMs`**；**`startTimeMs`** 规则为：  
   - 第一小节：从 **0** 到当前时间；  
   - 下一小节：从 **上一小节 end** 到当前时间。  
4. 状态栏会显示本节的 `start` / `end`（毫秒）。  
5. **务必保证**：按空格时，已经有一个画好的、尚未绑定的框在「队列」里；否则会提示 **「请先画框」**。

**推荐节奏**：画框 → 播放到合适位置 → 空格 → 再画下一框 → 再播放到下一位置 → 空格……

### 8. 使用「标注结果预览」

页面中部 **「标注结果预览」** 区域提供：

- **时间轴**：横向条表示整段时长，黄色块为各小节时间范围；**红线**为当前播放头。可点击**空白处**拖动跳转，或点击**某一黄色块**从该小节起点播放。
- **小节列表**：每行显示 `#序号` 与起止时间（`分:秒.厘秒`），**点击一行**即从该小节起点播放。
- **谱图上显示小节编号**：勾选后在右侧谱图矩形内绘制半透明底 + 白色描边数字。
- **高亮当前播放所在小节**：勾选后，列表与时间轴上当前时间落入的小节会高亮，谱图上该框为**蓝色描边**（随播放更新）。

### 9. 核对下方「预览 JSON」

- 检查 `measures` 里每个小节的 `rect`（x, y, w, h）是否为**原图像素坐标**（与图片分辨率一致）。  
- 检查 `startTimeMs` / `endTimeMs` 是否随音乐推进单调、合理。

### 10. 导出 data.json

1. 点击 **「导出 data.json」**，浏览器会下载文件。  
2. 将下载的 `data.json` 与**同一张谱面图**、**对应 MP3** 一起，通过 **管理页**（`/admin`）按 `songId` 上传发布，或自行放入服务器 `scores/<songId>/` 目录（需与后端约定一致）。

## 四、常见问题（排查）

| 现象 | 可能原因 | 处理 |
|------|----------|------|
| 选了图片却看不到图 | 布局宽为 0、脚本在 redraw 时除零、或格式不支持 | **强刷 + 重启服务**；已修复布局与 redraw 防护；换 JPG/PNG |
| 破图图标 / 音频 0:00 | **文件大小为 0** 或「导入」占位；`blob:` 在个别环境失效 | 页面会显示 **已选文件大小**；若为 **0 B**，把文件**复制到桌面**再选；文件名以 **Import-** 开头时尤需如此 |
| 选了音频不能播 | 未解码完成、或编码浏览器不支持 | 等待 **loadedmetadata** 或点播放；用 **标准 MP3**；看状态栏提示 |
| 在图上拖动没反应 | 图片未成功加载（natural 宽高为 0） | 先看到状态栏「图片已加载」再画框 |
| 按空格没反应 | 没有待绑定的框 | 先画矩形再按空格 |
| 空格触发页面滚动 | 焦点不在页面内 | 先鼠标点击谱图区域或页面空白，再按空格 |
| 时间与听感对不上 | 绑的是「当前时刻」为小节结束 | 在**该小节结束处**按空格；或导出后在 JSON 里手工微调毫秒 |
| 导出后 App 里框位置不对 | 上传的图与打点时用的图不是同一张或缩放不一致 | 保证管理端与 JSON 使用**同一原图**；`imageFile` 与真实文件名一致 |

## 五、与管理端、App 的配合（简要）

1. 打点小工具：本地生成 **`data.json`**（含 `measures`）。  
2. 管理页 `/admin`：上传 **谱面图**、**音频**、**data.json**（可选；不上传则服务端生成空 measures）。  
3. App：用 **Base URL + App Token** 拉列表并下载到本地 `scores/<id>/`，再离线练习。

## 六、快捷键与按钮一览

| 操作 | 方式 |
|------|------|
| 上传暂存（推荐） | 「上传到服务器」（需 Token；可选填后端根地址） |
| 画小节框 | 在右侧谱图上 **按住左键拖动** 拉出矩形 |
| 绑定时间与生成小节 | **空格**：把**当前播放时间**记为本小节 **结束时间**（毫秒），并与**最后一个未绑定的框**配对 |
| 标注预览 | 时间轴/列表点击跳转；可选谱图编号与当前小节高亮 |
| 播放/暂停 | 下方播放条 或 按钮「播放/暂停」 |
| 撤销 | 「撤销上一小节」（先删已绑定小节，再删未绑定框） |
| 导出 | 「导出 data.json」 |
